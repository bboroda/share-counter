# ShareCounter

Returns google plus shares for any url


## Installation

```
gem install share-counter
```


## Usage

```ruby
2.0.0-p353 :002 > require 'share-counter'
2.0.0-p353 :002 > url = 'http://makeshift.io'
 => "http://makeshift.io"
2.0.0-p353 :003 > ShareCounter.googleplus url
 => 29
```
